# translatePaper
## 概要
論文を翻訳するアプリ
pythonとgas(javascript)で実装

## 実行
以下のコマンドで`outputfile`に翻訳した結果を出力できる．
```
cd translate
python readFromPDF.py inputfile > outputfile
```
